# URL Analysis

## Objective

To inspect suspicious URLs embedded within phishing emails.

---

# Indicators Identified

## 1. Domain Spoofing
Fake domains imitated legitimate brands.

Example:
paypal-secure-login.xyz

---

## 2. Shortened URLs
Attackers used URL shorteners to hide malicious destinations.

---

## 3. HTTP Instead of HTTPS
Several phishing websites lacked secure HTTPS connections.

---

## 4. Redirect Chains
URLs redirected users through multiple suspicious websites.

---

# Conclusion

URL analysis identified malicious redirection techniques commonly used in phishing attacks.
